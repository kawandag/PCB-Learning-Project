# LED Learning PCB Project

## Overview

This project demonstrates the complete printed circuit board (PCB) design workflow using **LibrePCB 2.1.1**. The objective was to design a simple LED circuit while learning the fundamental stages of PCB development, from schematic capture to manufacturing output generation.

## Project Description

The circuit consists of a 2-pin power connector, a current-limiting resistor, and an LED connected on a single-layer PCB. This beginner-friendly project provides hands-on experience with the complete PCB design process and serves as a foundation for more advanced electronic designs.

## Features

- LED indicator circuit
- 330 Ω current-limiting resistor
- 2-pin power connector
- Through-hole (THT) components
- Single-layer PCB design
- 3D PCB visualization

## Software Used

- **LibrePCB 2.1.1**

## Skills Demonstrated

- Schematic Capture
- Electronic Component Selection
- Device and Footprint Assignment
- PCB Layout and Component Placement
- PCB Trace Routing
- Electrical Rule Check (ERC)
- Design Rule Check (DRC)
- Manufacturing File Generation

## Project Outputs

- Gerber Files
- Excellon Drill Files
- Bill of Materials (BOM)
- Pick-and-Place (PnP) File
- Schematic PDF
- Board Assembly PDF
- 3D PCB Model
- Project Archive

## Learning Outcomes

Through this project, I gained practical experience with:

- Creating electronic schematics
- Selecting appropriate electronic components
- Assigning PCB footprints to schematic symbols
- Designing and routing a PCB layout
- Verifying designs using ERC and DRC
- Generating manufacturing-ready files for PCB fabrication
- Visualizing the completed PCB in 3D
